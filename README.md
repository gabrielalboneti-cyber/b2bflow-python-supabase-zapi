# B2BFlow Python Supabase Z-API

Projeto em Python que busca contatos cadastrados no Supabase e envia uma mensagem personalizada pelo WhatsApp usando a Z-API.

A mensagem enviada é:

```txt
Olá, <nome_contato> tudo bem com você?
```

## Setup da tabela

No Supabase, crie a tabela `contatos` usando o SQL abaixo:

```sql
create table contatos (
  id bigint generated by default as identity primary key,
  nome text not null,
  telefone text not null
);
```

Para testar, você pode cadastrar alguns contatos assim:

```sql
insert into contatos (nome, telefone) values
('João', '5511999999999'),
('Maria', '5511888888888'),
('Pedro', '5511777777777');
```

O telefone precisa estar no formato internacional, com DDI e DDD.

Exemplo:

```txt
5511999999999
```

## Variáveis de ambiente

Crie um arquivo `.env` na raiz do projeto e preencha com suas credenciais:

```env
SUPABASE_URL=https://seu-projeto.supabase.co
SUPABASE_KEY=sua-chave-do-supabase

ZAPI_INSTANCE_ID=sua-instancia
ZAPI_INSTANCE_TOKEN=seu-token
ZAPI_CLIENT_TOKEN=seu-client-token
```

As chaves reais ficam somente no `.env`, não no código.

## Como rodar

Instale as dependências:

```bash
pip install -r requirements.txt
```

Depois execute:

```bash
python main.py
```

O projeto busca até 3 contatos na tabela `contatos` e envia a mensagem personalizada para cada um deles.
