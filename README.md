# B2BFlow Python Supabase Z-API

Projeto em Python que busca contatos no Supabase e envia uma mensagem personalizada via Z-API:

```txt
Olá, <nome_contato> tudo bem com você?
```

## Setup da tabela

No SQL Editor do Supabase, rode:

```sql
drop table if exists contatos;

create table contatos (
  id bigint generated by default as identity primary key,
  nome text not null,
  telefone text not null
);

alter table contatos enable row level security;

create policy "Permitir leitura de contatos"
on contatos
for select
to anon
using (true);

insert into contatos (nome, telefone) values
('João', '5511999999999'),
('Maria', '5511888888888'),
('Pedro', '5511777777777');

select * from contatos;
```

O telefone deve estar no formato internacional, com DDI e DDD:

```txt
5511999999999
```

## Variáveis de ambiente

Copie o arquivo `.env.example` para `.env`:

```bash
copy .env.example .env
```

Preencha o `.env` com suas credenciais:

```env
SUPABASE_URL=https://seu-projeto.supabase.co
SUPABASE_KEY=sua-chave-do-supabase
ZAPI_INSTANCE_ID=sua-instancia
ZAPI_INSTANCE_TOKEN=seu-token
ZAPI_CLIENT_TOKEN=seu-client-token
```

Não suba o arquivo `.env` para o GitHub.

## Instalar dependências

```bash
pip install -r requirements.txt
```

## Rodar

```bash
python main.py
```

Exemplo de saída:

```txt
Início da execução
Quantidade de contatos encontrados: 3
Enviando mensagem para João (5511999999999)
Sucesso no envio para João
Finalização
```
